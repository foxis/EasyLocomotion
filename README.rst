EasyLocomotion
--------------

Locomotion Arduino framework for: 

    - differential drive
    - N-legged robots
    - freespace(drone)
    - forward/inverse kinematics
    - linear algebra: matrix/vector algebra
    - various I2C devices(sensors, effectors, etc)
